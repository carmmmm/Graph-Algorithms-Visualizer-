Description

A graph is a pictorial representation of a set of objects. Object pairs in a graph are connected by links. The interconnected objects represented by points are called vertices, and the links that connect the vertices are edges.

In the first stage of the project, we will start by creating several vertices. A vertex can be represented by a labeled circle. In this project, we assume that you know a bit about the graphics class in Java, so roll up your sleeves and let's make our first program.

Objectives

Please follow the instructions below:

  1. Create a JFrame of the 800x600 size and name it Graph-Algorithms Visualizer;
  2. Create a graph (JPanel) in the JFrame and name it Graph;
  3. Place four vertices in four different corners inside the graph;
  4. A vertex should be represented by a JPanel. A vertex is merely a circle that has a label called Vertex ID;
  5. Create another JLabel inside each vertex. It should show the Vertex ID of that vertex;
  6. Four vertices (JPanel) and the labels (JPanel) should have the following names:
    • Top-left:
      • JPanel — Vertex 0
      • JLabel — VertexLabel 0
    • Top-right:
      • JPanel — Vertex 1
      • JLabel — VertexLabel 1
    • Bottom-left:
      • JPanel — Vertex 2
      • JLabel — VertexLabel 2
    • Bottom-right:
      • JPanel — Vertex 3
      • JLabel — VertexLabel 3
